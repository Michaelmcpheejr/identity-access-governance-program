# Privileged Access Risk Controls

## Definition

Privileged access includes roles or permissions that can:

- Modify identity policies
- Access sensitive data
- Change security configurations
- Bypass standard restrictions

## Baseline Requirements

- MFA enforced for all privileged access
- Group-based RBAC assignments only
- Logging of privileged actions
- Monthly privileged access reviews

## Exceptions

Exceptions are allowed only with:

- Written business justification
- Compensating controls
- Executive risk acceptance approval
