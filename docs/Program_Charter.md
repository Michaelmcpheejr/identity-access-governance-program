# Program Charter: Identity & Access Governance Program

## Purpose

This program ensures identity and access controls are consistently governed to reduce unauthorized access, privilege misuse, and lifecycle gaps across enterprise systems.

## Scope

Applies to:

- Azure Entra ID identity governance
- Microsoft 365 administrative roles
- SaaS applications integrated through SSO and RBAC groups

## Control Objective

Implements Zero Trust aligned governance through:

- Tiered admin privilege separation
- Continuous privileged monitoring
- Periodic access certification
- Lifecycle enforcement for joiners, movers, and leavers

## Ownership Model

- Enterprise Risk provides governance and risk appetite
- IAM executes provisioning, reviews, and enforcement
- Security monitors privileged activity and validates controls
